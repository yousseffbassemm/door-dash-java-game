# DooR DasH – Turn-Based Java Board Game

A turn-based board game inspired by strategic movement and card mechanics.

## Features
- Player movement system
- Card mechanics
- Energy system
- Win conditions
- Object-oriented design

## Technologies
- Java
- OOP Principles

## What I Learned
- Inheritance
- Polymorphism
- Encapsulation
- Debugging
- Team collaboration

## How to Run
1. Clone repository
2. Open in IntelliJ / Eclipse
3. Run Main.java
